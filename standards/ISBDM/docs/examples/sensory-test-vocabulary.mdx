# Sensory Specification Test Vocabulary

This example demonstrates the VocabularyTable component using the `sensory-test.csv` file, which includes comprehensive SKOS properties including examples, change notes, history notes, and editorial notes.

## Features Demonstrated

- **CSV Data Loading**: Vocabulary terms loaded from external CSV file
- **Multilingual Support**: Terms available in English, French, and Spanish
- **Rich SKOS Properties**: Includes notation, examples, change notes, history notes, editorial notes
- **Expandable Details**: Click the "+" button to see additional properties
- **Language Switching**: Use the language selector to switch between languages
- **Search/Filter**: Use the search box to find specific terms

## Basic Example

<VocabularyTable 
  vocabularyId="sensory-test"
  title="Sensory Specification Vocabulary (Test Data)"
  description="Types of sensory content specifications for bibliographic resources"
  scopeNote="This vocabulary provides standardized terms for describing the sensory characteristics of content"
  prefix="sensoryspec"
  uri="https://iflastandards.info/ns/isbd/terms/contentqualification/sensoryspecfication"
  isDefinedBy="https://iflastandards.info/ns/isbd/terms/contentqualification/sensoryspecfication"
  csvFile="/data/CSV/sensory-test.csv"
  showTitle={true}
  defaultLanguage="en"
  availableLanguages={["en", "fr", "es"]}
/>

## Technical Details

**CSV File**: `/static/data/CSV/sensory-test.csv`

**SKOS Properties Available**:
- `skos:prefLabel` (English, French, Spanish)
- `skos:definition` (English, French, Spanish)  
- `skos:altLabel` (French, Spanish)
- `skos:notation` (English, French, Spanish)
- `skos:example` (English, French, Spanish)
- `skos:changeNote` (English, French, Spanish)
- `skos:historyNote` (English, French, Spanish)
- `skos:editorialNote` (English, French, Spanish)

**Key Features**:
- Rich multilingual content with comprehensive SKOS properties
- Expandable details for additional properties
- Demonstrates editorial workflow with change notes and history
- Shows best practices for vocabulary maintenance and documentation
