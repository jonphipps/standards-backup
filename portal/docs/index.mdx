## Getting Started

If you're a member of an IFLA Review Group or an editor:

1. Navigate to your standard's documentation using the links above
2. Use the editing tools to propose changes
3. Submit your updates for review

For more information on using this workspace, see our [contributor guidelines](/contribute).
